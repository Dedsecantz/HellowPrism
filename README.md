## What is Git

Git is a distributed version control system used to track changes in code.  
It helps developers collaborate and maintain different versions of a project.

Key features:
- Tracks file changes
- Allows multiple developers to work together
- Lets you revert to previous versions

---

## How to Push Code to GitHub

Follow these steps:

1. Initialize Git in your project
git init
2. Add files to staging
git add.
3. Commit the files
git commit -m "Initial commit"
4. Connect your project to GitHub
git remote add origin https://github.com/username/repository-name.git
5. Push the code
git push -u origin main

Markdown is a lightweight markup language used to format text using simple symbols.  
It is commonly used for README files and documentation.
